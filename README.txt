The makefile is included, the code should be compiled by simply type:
>> Make
To start a server listening to a specific port_number, using:
>> ./rfss port_number
Some information will show up on terminal:

localhost name: xinu01.cs.purdue.edu
PACKAGE_SIZE = 512
tryint server on: 3456
Please input your option... Type help for assistance...

Then it's ready to receive commands, (case insensitive)

When exiting, the main thread will sleep for 2 seconds to yield the schedule for others.
